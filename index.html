实习介绍

企业简介

中车唐山机车车辆有限公司（简称中车唐山公司）是中国中车股份有限公司核心子企业，始建于1881年，是中国轨道交通工业的发轫之地。公司主要从事轨道交通装备的研发、制造、检修、服务以及数字产业化、新能源等战略性新兴产业。

拥有唐山、天津、泉州三大制造基地，职工近万人，是中国第一台蒸汽机车“中国火箭”号、第一辆御用客车“銮舆”及首列时速350公里高速动车组的诞生地。

项目简介：智联未来·零碳出行

以某生态公园为设计场景，围绕“智联化”与“零碳化”主线，设计一套高架旅游观光交通系统，包含：

  • 观光载具（空中轨道列车）
  • 车站与站点
  • 线路规划
  • 周边配套环境
  • 智能交互系统（APP + 自助服务机）

目标：打造国家级体育露营示范项目，融合运动、休闲、商业、生态于一体。

实习内容及成果

第一周（7.7–7.13）

参加“2025天津大学国际暑期设计工作坊”，担任小组组长，带领来自英国代尔夫特大学与天津大学的成员完成“智联未来·零碳出行”方案设计。

团队从4个小组中脱颖而出，荣获“优胜小组”称号。

完成内容：空中轨道线路规划、载具外观与尺寸设计、高架桥结构、玻璃观景平台、整体视觉风格（如“白色邮轮”概念）。

第二周（8.12–8.17）

深化项目分工，我独立负责智能交互系统设计。

工作内容：

  • 基于 JTBD（Jobs-To-Be-Done）理论分析用户需求
  • 识别五大用户群体：本地居民、游客、学生、运动爱好者、特殊需求人群
  • 拆解核心任务：导航、购票、紧急求助、活动信息、无障碍服务等
第三周（8.18–8.24）

启动 APP 端详细设计，构建五模块架构：

  1. 主页（公园资讯、快捷入口）
  2. 出行（地图、路线规划、步行导航）
  3. 乘车（二维码扫码、车辆信息）
  4. 专区（运动、宠物、露营、音乐节等垂直场景）
  5. 我的（账户、设置、紧急联系）

采用卡片化布局 + 统一主题色（蓝白科技风），确保美观与易用性。

第四周（8.25–8.31）

根据导师与中车专家反馈，迭代优化 APP 设计。

同步完成自助服务机端设计：

  • 操作逻辑：待机 → 语言选择 → 主页 → 功能（地图指引 / 紧急求助）
  • 界面简洁，仅保留核心功能，避免信息过载
  • 视觉风格与 APP 保持一致，确保跨终端体验统一
最终成果：一套完整的“云道 CloudRoad”智能观光交通交互系统设计方案，可直接用于作品集展示。

实习收获及体会

本次实习是我学生生涯中一次意义非凡的深度学习与成长之旅。

  • ✅ 跨文化协作能力提升：与国际团队高效合作，克服语言与专业差异。
  • ✅ 系统化设计思维：从用户需求出发，贯穿调研→分析→原型→迭代全流程。
  • ✅ 工业级项目经验：在真实企业场景中接受专家评审，理解产业落地逻辑。
  • ✅ 技术与人文融合：在“零碳出行”理念下，探索科技如何服务生态与人文。

特别感谢中车唐山公司提供宝贵平台，以及天津大学、代尔夫特大学伙伴们的信任与支持。这段经历不仅锤炼了我的专业能力，更让我看到设计在推动可持续城市交通中的巨大潜力。
